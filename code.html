Other codes and tools I recommend

  • xarray for python: Remember last time you shared a 2d dataset (e.g., something that depends on mass and redshift) with someone? You probably sent the 2d dataset as an ascii or .npy file and then told the other person what the labels/coordinates and units were. And most likely, there was a misunderstanding somewhere. Use xarray (it actually works for N-dimensional data)!
  • Stop wasting your time maintaining a .bib file, use adstex!
  • scholarly, a python interface to Google Scholar
